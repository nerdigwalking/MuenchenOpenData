Impressum

Dieses Dashboard ist ein privates Demonstrationsprojekt zur Visualisierung öffentlicher Open-Data-Quellen der Landeshauptstadt München. Es besteht kein offizieller Zusammenhang mit der Stadt München.

Datenschutz

Dieses Dashboard lädt Daten direkt aus öffentlichen APIs. Es werden keine personenbezogenen Daten gespeichert. Die Kartenkacheln werden von OpenStreetMap-Servern bezogen; dabei gelten deren Datenschutzhinweise.

Haftungsausschluss

Die Daten werden ohne Gewähr auf Vollständigkeit oder Aktualität bereitgestellt. Für die Inhalte der verlinkten externen Quellen wird keine Haftung übernommen.
